# File manager

CLI tool for files manipulating

## Start tool

Type in console:
```bash
npm run start -- --username=your_username
```

Example:

```bash
npm run start -- --username=jane_doe
```

## Commands
| command | action    |
|---------|-----------|
| `.exit` | exit tool |

## Exit

`Ctrl+C` or type in console with active tool `.exit` command
